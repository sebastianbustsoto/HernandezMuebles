Hernández Muebles - Plataforma de Gestión de Cotizaciones y Pedidos
Rubro: E-Comerce

Descripción del Proyecto

Hernández Muebles es una plataforma web diseñada para optimizar la gestión de preventa y producción de muebles a medida. La aplicación permite a los clientes enviar solicitudes personalizadas con descripciones técnicas e imágenes de referencia, mientras que el administrador gestiona el flujo completo desde la cotización hasta la entrega final del producto.

**Tecnologías Utilizadas
**
Frontend: Next.js 15 (React 18), Tailwind CSS 4

Backend: Next.js API Routes

Base de Datos: PostgreSQL (Supabase)

Almacenamiento: Supabase Storage

Autenticación: Email + OTP (EmailJS)

Documentos: PDFKit

Control de Versiones: Git + GitHub
